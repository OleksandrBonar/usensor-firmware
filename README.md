# usensor-firmware
Universal sensor

## Features
 - Display (SSD1306)
 - MQTT (over WiFi)
 - Temperature (SHT31)
 - Humidity (SHT31)
 - Illuminance (BH1750)
 - UV ([GUVA-S12SD](https://github.com/OleksandrBonar/usensor-firmware/main/README-GUVA-S12SD.md))
 - Motion (HC-SR501)

## Hardware
All devices based on ESP8266
